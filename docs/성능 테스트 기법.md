# 성능 테스트 기법



1. 부하테스트 툴 선정



- Wrk : 간단한 서버 부하 테스트 
- Vegeta : 초당 몇 개의 부하를 버티는지 테스트 
- Gatling : 높은 성능 , 시나리오 작성, html 보고서 (코드 작성 필요)
- JMeter : 다양한 기능이나 플러그인이 강점 / 빠른 시나리오 작성 기능
- nGrinder : 시나리오, 시나리오 가중치 테스트 불가 , 개별 트랜젝션 TPS 측정에 중점, Thread 기반을 구현되어 있어 성능과 동시성에 대한 제한이 있음
